# PROJECT TIC-TAC-TOE

## Description:
This project is an exercise of using IIFEs, modules, and closures in JavaScript. The main idea is on the game logic, localizing most of the piece of the code for better organization.

## Skills used:
The skills used in making this project possible are:
- Basic HTML and CSS manipulation
- Proficiency in using CSS Grid and CSS Flexbox
- Basic JavaScript DOM Manipulation
- JavaScript Module Pattern
